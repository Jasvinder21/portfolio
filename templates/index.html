Anime Recommendation System

Developed a Flask-based web application with API integration to recommend anime based on user-provided descriptions, leveraging NLP and recommender systems. Built a text embedding system using Sentence Transformers and FAISS for efficient similarity search.

AI Document Assistant Using Llama3

Developed a retrieval-augmented generation (RAG) system using Llama3 and Streamlit to query PDF documents interactively. Implemented PyPDF2 for text extraction and FAISS with LlamaIndex for vector search.

AI Content Summarizer
